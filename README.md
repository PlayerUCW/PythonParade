# PythonParade

Набор инструментов для управления проектами для конкурса "Python-парад".

## ТЗ

Набор состоит из инструментов, выполненных в виде оконных приложений Windows Forms. Используемый язык - C#.

Для проведения конкурса требуются возможности:

- Регистрировать проекты в системе
- Голосовать за проекты
- Модерировать проекты и голоса
- Оформлять связанные документы

Каждый нструмент набора имеет свою область применения:

- Редактор проектов. Применяется учащимися на этапе регистрации проектов на конкурс. Позволяет просматривать список проектов, регистрировать новые проекты и вносить в них правки;
- Опрос. Применяется посетителями конкурса во время его проведения. Позволяет голосовать за понравившийся проект, имеет несколько номинаций;
- Центр управления. Применяется администратором на этапах регистрации проектов на конкурс, проведения конкурса и подведения итогов конкурса. Позволяет модерировать проекты и голоса, указывать требуемые правки к проектам, подсчитывать голоса, определять проекты-победители конкурса и готовить сопутствующие документы.

## Документация

Набор находится в разработке, реализованы следующие инструменты:
- Меню просмотра проектов
- Редактор проектов

### Классы

- `Student` - класс для удобной работы с ФИО учащегося
- `Project` - класс проекта (его регистрационной записи)

Формы:

- `AppChoiceForm` - окно выбора инструмента
- `ProjectsForm` - окно просмотра проектов
- `EditorForm` - окно редактора проектов

### Класс Student

- `Equals` - сравнивает два `Student`. Также определены операторы `==` и `!=`.
- `GetFullName` - возвращает строку с ФИО учащегося в требуемом формате
- `GetListViewItem` - возвращает `ListViewItem` с ФИО учащегося в требуемом формате

### класс Project

- `AddAuthor` - добавляет автора в формате `Student`. Используется при создании проекта
- `AuthorsText` - возвращает строку с авторами проекта для размещения на элементе `Label`
- `StatusText` - возвращает строку, соответствующую статусу проекта
- `StatusColor` - возвращает цвет, соответствующий статусу проекта
- `GetListViewItem` - возвращает `ListViewItem` с названием и статусом проекта, `ListViewItem.Tag` указывает на этот проект
- `GetAuthorsListViewItems` - возвращает список `ListViewItem`, описывающих авторов проекта
